Torre Windows - Bibliotecas Python

Aqui estão algumas bibliotecas Python recomendadas para trabalhar com ficheiros, periféricos, som e imagem no Windows:

Bibliotecas para Ficheiros:

  • os - Interação com o sistema operativo, manipulação de ficheiros e diretórios.
  • shutil - Operações de ficheiros e cópias de diretórios.
  • pathlib - Manipulação de sistemas de ficheiros usando uma interface orientada a objetos.

Bibliotecas para Periféricos:

  • pySerial - Comunicação via porta serial (USB, RS232, etc.).
  • hidapi - Interação com dispositivos HID (Human Interface Devices).

Bibliotecas para Som:

  • pyaudio - Captura e reprodução de som.
  • wave - Leitura e escrita de ficheiros de som no formato WAV.
  • pygame.mixer - Módulo de áudio da biblioteca Pygame.

Bibliotecas para Imagem:

  • Pillow - Manipulação de imagens (substituta da PIL).
  • OpenCV - Processamento de imagens e visão computacional.
